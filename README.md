# Volcano Staking

> High-yield staking with volcanic eruption mechanics

## Features

- Stake tokens
- Eruption events
- Compound rewards
- Tier system

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/bomniop/volcano-staking.git
cd volcano-staking
npm install
npm run dev
```
